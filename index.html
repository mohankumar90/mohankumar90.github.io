Convex Hull Based Detection of Overlapping Red Blood Cells in Peripheral Blood Smear Images

  • Conference : WACBE 2015
  • Date : 06/07/2015
  • Used Technologies : Java & ImageJ libraries

The Segmentation of Red Blood Cells (RBCs) in blood smear images to obtain their count is often the first step in the diagnosis of various pathological conditions. Although several procedures have been devised for this task, a majority of them suffer from performance degradation due to the overlapping of cells. Various researches have been carried out to split these overlapping cells. This project aims to find the concavity points in the overlapping RBCs’ contours. This approach's dip point identification is based only on the convex hull of the overlapping cell. The contours of the concave regions are analyzed from the perspective of the centroid.

ARTSens - Android version

  • Duration : Jun 2016 - Nov 2016
  • Used Technologies : Java, Android Studio IDE

ARTerial Stiffness Evaluation for Non-invasive Screening (ARTSENS) is the project for measuring arterial stiffness of the patients, using ultrasound signals and it's reflections. Android version of this project is on development stage and we have worked on Data acquisition block and Plotting graph block. Ultrasound probe, which sends 5000 data-sets per cycle with the average of 25 cycles/sec using Android USB device/host serial communication method. From the device (ultrasound probe) to host (android device), we need to receive the data continuously and processing it with the Digital Signal Processing algorithms, and continuing blocks of artery wall detection and wall tracking. Still this project is on under development . I just worked on this project up-to data acquisition and plotting those values into graph, without noise filter.

  • Used Technologies : Java, Android Studio IDE

Android version of this application started to develop in Andriod Studio, and the app has challenging requirements as follows:

  2. Camera should have capabilities of capture RAW (.DNG extension) image &
